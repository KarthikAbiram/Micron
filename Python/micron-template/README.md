# Setup
Install [uv](https://xtldr.com/uv/) and [make](https://xtldr.com/make/)

To create virtual environment and install dependencies:
```
make install
```

# Run Source
To run server:
```
make run-server
```

To run client, open another terminal and run:
```
make run-client
```

# Build
To build both server and client:
```
make build
```

For more details, please refer to https://github.com/KarthikAbiram/Micron/tree/main/Python/micron-template
